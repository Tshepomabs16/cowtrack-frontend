# CowTrack Frontend

CowTrack Frontend is the user interface for the CowTrack system, created to help users interact with the livestock tracking platform in a simple and accessible way.

## Features
- Frontend interface for CowTrack
- User interaction with tracking data
- Structured layout for livestock information
- Works alongside the backend system

## Tech Stack
- JavaScript
- Frontend development
- HTML
- CSS

## Purpose
This project was developed as the frontend side of the CowTrack application to provide a usable interface for livestock tracking and management.
